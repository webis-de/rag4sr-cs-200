| Inclusion criteria | Exclusion criteria |
|---|---|
| · Original peer reviewed published manuscripts | · Solely investigated gait analysis for clinical purposes |
| · Aimed at a sport-specific movement or skill | · Solely investigated every day or non-sport-specific locomotion i.e., walking downstairs |
| · Used IMUs and/or computer vision input datasets for model development | · Solely investigated player field positional tracking methods using data such as X, Y coordinates or displacement without any form of sport-specific skill detection and classification associated to it |
| · Investigated as an in-field application of the technology to the sporting movement | · Used ball trajectory and audio cue data as the major determinant for event detection |
| · Defined clear data processing and model development methods inclusive of machine or deep learning algorithms for semi-automated or automated movement recognition | · Data collection conducted within a laboratory setting under controlled protocol |
| · Published as full-length studies written in English | · Data processing pipelines or recognition model development methodology not clearly defined |
| | · Review studies |
