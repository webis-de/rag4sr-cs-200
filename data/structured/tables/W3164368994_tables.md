| Metric | Formula |
|--------|---------|
| TP | ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*jj*</sub> |
| TN | ∑<sub>*f*=1</sub><sup>*k*</sup> *n*<sub>*ff*</sub> |
| FP | ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*ij*</sub>, *i* ≠ *j* |
| FN | ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*ji*</sub>, *i* ≠ *j* |
| Precision | TP/(FP+TP) = ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*jj*</sub> / (*n*<sub>*ij*</sub>+*n*<sub>*jj*</sub>), *i* ≠ *j* |
| Recall or Sensitivity | TP/(FN+TP) = ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*jj*</sub> / (*n*<sub>*ji*</sub>+*n*<sub>*jj*</sub>), *i* ≠ *j* |
| FPR | FP/(FP+TN) = ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*ij*</sub> / (*n*<sub>*ji*</sub>+*n*<sub>*jj*</sub>), *i* ≠ *j*, *i* ≠ *f* |
| FNR | FN/(FN+TP) = ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*ij*</sub> / (*n*<sub>*ji*</sub>+*n*<sub>*jj*</sub>), *i* ≠ *j* |
| Specificity | TN/(FP+TN) = 1-FPR = ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*ff*</sub> / (*n*<sub>*ij*</sub>+*n*<sub>*ff*</sub>), *i* ≠ *j*, *i* ≠ *f* |
| Accuracy | (TP+TN)/(FP+FN+TP+TN) = ∑<sub>*j*=1</sub><sup>*k*</sup> (*n*<sub>*ff*</sub>+*n*<sub>*jj*</sub>) / (*n*<sub>*jj*</sub>+*n*<sub>*ff*</sub>+*n*<sub>*ji*</sub>+*n*<sub>*ij*</sub>), *i* ≠ *j*, *i* ≠ *f* |
| AUC | (Recall+Specificity)/2 = ½ × (TP/(TP+FN) + TN/(TN+FP)) |
| MCC | (TP×TN−FP×FN) / √[(TP+FP)(TP+FN)(TN+FP)(TN+FN)] |
| Kappa Coefficient | (*P*<sub>0</sub>−*P*<sub>*c*</sub>)/(1−*P*<sub>*c*</sub>) |
| DSC | 2×Precision×Recall/(Precision+Recall) = 2×TP/(FP+FN+2TP) = ∑<sub>*j*=1</sub><sup>*k*</sup> 2×*n*<sub>*jj*</sub> / (*n*<sub>*ij*</sub>+*n*<sub>*ji*</sub>+2*n*<sub>*jj*</sub>), *i* ≠ *j* |
| IoU | TP/(FP+FN+TP) = ∑<sub>*j*=1</sub><sup>*k*</sup> *n*<sub>*jj*</sub> / (*n*<sub>*ij*</sub>+*n*<sub>*ji*</sub>+*n*<sub>*jj*</sub>), *i* ≠ *j* |
